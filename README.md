# GameTracker

GameTracker es una aplicacion para descubrir videojuegos, gestionar tu biblioteca, puntuar juegos, escribir reviews, seguir a otros usuarios y consultar estadisticas de juego.

## Documentación

La documentacion del proyecto se encuentra centralizada en:

- [docs/README.md](./docs/README.md)

Desde ahi puedes acceder a:

- roadmap del producto
- estructura del proyecto
- overview de APIs
- modelo de datos
- decisiones de arquitectura
- guia de contribucion
- changelog
- guia de despliegue

```text
gametracker/

```

## Estado actual

Actualmente el proyecto cuenta con:

- frontend en React + Vite
- servidor Express dentro de `frontend/`
- backend FastAPI dentro de `backend/`
- integracion con IGDB/Twitch
